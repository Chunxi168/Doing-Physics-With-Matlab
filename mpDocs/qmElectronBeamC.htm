STEP POTENTIAL  E > U0

 

In this article, we will consider what happens when an electron beam is incident upon a step potential barrier where the height of the barrier is less than the particle’s total energy (figure 1).

In classical physics, a particle of total energy E travelling in the direction of increasing x where , will suffer an impulsive force  at the point . But, the impulse will only slow the particle, and it will continue moving in the direction of increasing x. Its total energy E will remain constant but its momentum p will be reduced.

Since the potential energy function is independent of time, to determine the motion of the electron according to quantum mechanics, we must find a solution of the time independent Schrodinger equation, For a step potential, the Schrodinger equation can be solved exactly using analytical methods or approximation solutions found using numerical methods such as the finite difference approximation. It is often much easier to solve ordinary differential equations numerically, rather than doing lots of algebra to get an analytical solution. Also, for more realistic potentials energy functions, there are no analytical solutions.

 

The electron is not a classical particle or a classical wave. The behaviour of an electron is simply described by the wavefunction  which is simply a mathematical function which has no physical meaning. So, what does the wavefunction represent?

 

For our electron beam, the wavefunction  is not related to individual particles, but we consider the scattering of a steady intense beam of particles, each particle having energy E. It is not possible to predict the exact behaviour of any individual particle, but if the incident beam is sufficiently intense, the scattering can be described in terms of the refection and transmission coefficients which can be determined from the wavefunction.

The Script qmElectronBeamCA.m is used to model the scattering from the step potential using analytical methods.

 

The model parameters are changed in the INPUT section of the Script.

The default value of A is 1x108, so the incident linear density of the beam is set at 1x1016 particles/m.

 

% INPUTS ==============================================================

%  Energy of particles  [250 eV];

   Es = 4*200/3;

 

% Step Potential: height of step E > U0   [200 eV]

  U0s = 200;

 

% Amplitude of incident wave  [10^8]

  A = 10^8;

  

% X axis  [-0.5  0.5 nm]

  xMin = -0.6;

  xMax = 1;

  

% Grid points and indices for region x < 0 and x > 0

  N = 5501;            % must be an odd number

 

From the input parameters, we can easily calculate parameters which describe the time evolution of the system.

 

% MODEL PARAMETERS  ================================================== 

% Region 1 and Region 2: first and second elements of a vector 

 

% Angular frequency  omega  [1/s] 

  w = E/hbar;

% Frequency  [Hz]

  f = 2*pi/w;

% Period T [s]

  Tp = 1/f;

% Time grid

  nT = 201;

  t = linspace(0,1*Tp,nT);

  psiT = exp(-1i*w*t); 

 

% Kinetic Energy  [eV and J]

  KEs(1) = Es;

  KEs(2) = Es - U0s;

  KE = e.*KEs;

 

% Momentum  [kg.m/s];

  p = sqrt(2*mE*KE);

 

% Velocity  [m/s]

  v = p./mE;

 

% Wavelength  lambda  [m and nm]

  wL = h./p;

  wLs = wL./X;

 

% Wave number  [1/m]

  k = p./hbar;

 

% Amplitudes: B reflected wave / C transmitted wave

  B = ( k(1)-k(2)) / (k(1)+k(2 ))*A;

  C = 2*k(1) /( k(1)+k(2) )*A;

 

% Reflection and transmission coefficients

  R = B^2/A^2;

  T = C^2/A^2;

  

% Eigenfunction  psi(x)

  psi = zeros(N,1);

  for c = 1 : N

    psi(c) = A*exp(1i*k(1)*x(c)) + B*exp(-1i*k(1)*x(c));

    if x(c) > 0

      psi(c) = C.*exp(1i*k(2)*x(c));

    end

  end

 

% Probability density  [1/m]

   psiS = conj(psi);

   PD = conj(psi).*psi;

   psi_max = max(real(psi));

  

% Number of particles in cylinder: x < 0 num1 and x > 0 num2

  num(1) = simpson1d(PD(1:N2(1))',x(1),x(N2(1)));

  num(2) = simpson1d(PD(N2(1):N2(end))',x(N2(1)),x(N2(end)));

 

% Number density

  n(1) = num(1)/abs(x(1));

  n(2) = num(2)/abs(x(N));

     

% Probability current   [1/s]

  J = psiS.*gradient(psi,dx) - psi.*gradient(psiS,dx);

  J = -(1i*hbar/(2*mE)) .* J;

  Jnet = mean(J);

 

  Jinc   = v(1)*A^2;

  Jrefl  = -v(1)*B^2;

  Jtrans = v(2)*C^2;

 

  J1 = Jinc + Jrefl;

  J2 = Jtrans;

Note: the reflection coefficient is greater than zero and less than one 0 < R < 1. So, some particles must be reflected from the step barrier. A classical particle would never be reflected – classical concepts fail in this instance. However, a classical wave would be reflected. The only conclusion we can draw is that electrons behaviour more like a classical wave than a classical particle. For the case when E < U0, all particles were reflected R = 1.

 

The number of particles is found by integrating the probability density in region 1 (x < 0) and region 2 (x > 0). The integration is done using Simpson’s rule (need to download the function simpson1d.m)

 

% Probability density  [1/m]

   psiS = conj(psi);

   PD = conj(psi).*psi;

   psi_max = max(real(psi));

 

% Number of particles in cylinder: x < 0 num1 and x > 0 num2

  num2 = simpson1d(PD(N2(1):N2(end))',x(N2(1)),x(N2(end)));

 

The probability current is calculated in two different ways.

% Probability current   [1/s]

  J_AC = mean(J);

When the beam encounters the step barrier, the matter wave is partially reflected and partially transmitted. The incident wave and the reflected wave interfere with each other producing a partial standing wave. The transmitted wave propagates in the direction of increasing x. Figure 2 shows an animation of the real and imaginary parts of the wavefunction . The animation clearly shows the partial standing wave and the traveling transmitted wave.

 

Fig.2.   The real and imaginary parts of the wavefunction  as functions of both space and time. For x < 0 a partial standing wave is created. For x > 0, a travelling wave propagates in the direction of increasing x.

Fig. 3.   Top graph shows the potential energy function and the total energy of a particle. The middle graph 1 shows the real and imaginary parts of the eigenfunction. Middle graph 2 shows the probability density function. The area under a section of width  gives the number of particles within the length . The bottom graph shows the probability flux. The probability flux is independent of both space and time.

 

In the region x < 0, the wavefunction is a combination of the incident wave travelling to the right (amplitude A), and a reflected wave (amplitude A/3) moving to the left. As the amplitude of the reflected wave is always smaller than the amplitude of the incident wave, the two waves cannot combine to produce a pure standing wave, is something like a combination of a travelling wave and a standing wave. In the region x > 0, the wavefunction is a pure travelling wave of amplitude (4/3)A travelling to the right, and so the probability density is constant. The bottom graph shows that the probability current is independent of x. This is because there must be a constant flow of particles moving in the direction of increasing x since there is no creation or annihilation of particles. In this example, k1 = 2k2, hence max(PD) / min(PD) = 4 where PD is the probability density (see plot). For the amplitudes, B/A = 1/3 and C/A = 4/3. These values can be calculated in the Matlab Command Window.

 

The probability flux incident  upon the potential step is split into a reflected flux  and a transmitted flux such that  such that .  At the step, the particle does not vanish or split. A particle hitting the step will either be reflected or transmitted. For many particles impacting the step, the particles will go one way or the other, the probability of a particle going it the left is measured by R and the probability of a particle going to the right is measured by T.

If the direction of the incident wave is reversed and travels in the direction of decreasing x, then the values of R and T do not change as shown by inspecting equation 17. The reflection occurs at the step simply because of the discontinuity in the value of the potential. This is called the reciprocity property, and is a property of all waves. Again, we see that our atomic sized particles move in a wavelike manner.

We need to solve for both the real and imaginary parts of the eigenfunction. The function for the solving the Schrodinger equation is

function du = FNode(x,u)

 global E K

 

  U = pot(x);

  U = U*1.6e-19;

  y = u(1);

  ydot = u(2);

  du = zeros(2,1);

  du(1) = ydot;

  du(2) = -K*(E-U)*y;

 end

To get the eigenfunction psi, we need to flip the values of the eigenfunction u return by the ode45 function. Then, we normalise both the real and imaginary parts to 1 and then multiply by a 1.333x108 to set the amplitude of both the real and imaginary parts to 1.333x108. This sets the amplitude of the incident wave to 1.0000x108 which is the value used in the model using analytical the solution.
